# BrewApps-Assignment

# API endpoints and their usage
1. create - It is used for creating book in our database
2. getBook - It is used for accessing all the books which is present in database
3. getBookById - It is used for accesing a perticular book using the Id
4. updateBook - It is used for updating the som information of book which is already present in a database using its Id
5. deleteBook - It is used to delete a book from database using its Id

# step 1 
    run "npm i" on the terminal for download the dependency

# step 2
    run "npm start" on terminal for start the server

# step 3
    check all the api's using postman or other
